
#JADWAL SHALAT
Mengaktifkan Jadwal Shalat :
	- Masuk kedalah halaman Administrator OpenSID
	- Pilih Menu : Admin Web - Tema
	- Pencet tombol pengaturan Tema Batuah
	- Pada form "Tampilkan Jadwal Shalat", pilih "Aktif"
	- Pada "Kota untuk Widget Jadwal Shalat" pilih Kota/Kab
	- Klik tombol "Simpan"

#Warna Tema
Memilih warna tema default :
	- Masuk kedalah halaman Administrator OpenSID
	- Pilih Menu : Admin Web - Tema
	- Pencet tombol pengaturan Tema Batuah
	- Pada form "Template Warna Default Tema", pilih template warna
	atau, dengan cara manual :
	- Pada form "Warna Tema Random" pilih "Aktif"
	- Kemudian pilih warna pada bagian kota warna pada form :
	  "Warna Solid 1" s/d "Warna Soft 2"
	- Klik tombol "Simpan"
	Note : Perhatikan fungsi masing-masing form
